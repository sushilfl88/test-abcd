Response Options

Enable Response - Enables response.

Title - Enter title for new response.

Answer - Enter response text. BB code may be used to enhance formatting. More info on BB code here.

Preview Message

Once you have typed some text into the textarea you can use this option to preview the post. This will format any BB code used. This screenshot shows the YouTube BB formatting when the preview option is used.



To clear the preview click on the text area text or use the close button.

Departments

Click the Departments tab to set which departments this response applies to. If no departments are set, it will default to all departments. This determines which responses to make available on a ticket page by the reply box. If you always select ALL departments, then ALL responses will display, resulting in slower load times. It is recommended you group responses by department, but this is optional.
