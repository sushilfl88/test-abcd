Overview

Enables various batch tools and cleanup operations. Please read carefully. Click example images to display full size version in new tab/window.

Purge Options - Purge Closed Tickets X Days Old

Options

Days Old - Days old a ticket must be for purging.

Departments - Check departments to filter by department.

Clear Attachments - If you wish to also clear attachments, check box

Purge Options - Clear ticket attachments only X days old

Options

Days Old - Days old an attachment must be for purging.

Departments - Check departments to filter by department.

Purge Options - Clear accounts with NO tickets X days old

Options

Days Old - Days old an account must be for purging. Only accounts with NO tickets at all are purged.

Send E-Mail Notification About Account Removal - If you wish to send notification of account closure, check box.

Purge Options - Batch Enable/Disable

Options

Areas - Check areas you wish to perform enable or disable.

Enable Selected - Enables selected areas.

Disable Selected - Disables selected areas.

Purge Options - Global Password Reset (ONLY SEEN BY GLOBAL ADMIN (ID:1))

Options

All Visitor Accounts - If checked, resets all visitor account passwords.

All Support Team Accounts (Excluding global admin) - If checked, resets all support staff passwords, EXCEPT global admin (ID:1).

Include Disabled Accounts - If checked, will also reset disabled accounts.

Send Email - If checked, enables option to send email notification.

Message - Only visible if "Send Email" is checked. Enter message. <-{separater}-> denotes mail separater for header/footer. More info on the email template structures here.

Mail Tags - Adds tags to personalise messages. Tags must NOT be renamed.
