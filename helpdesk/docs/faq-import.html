Overview

Enables batch importing of FAQ questions. Please read carefully. Click example images to display full size version in new tab/window.

Screenshot

Import Options

Locate CSV File - Locate local import file on computer. File doesn`t have to be a .csv file, but must be formatted correctly. See below.

Lines to Read - Must be greater than the longest line (in characters) to be found in the CSV file (allowing for trailing line-end characters). It became optional in PHP 5. Omitting this parameter (or setting it to 0 in PHP 5.0.4 and later) the maximum line length is not limited, which is slightly slower. NO commas or period symbols. Examples: 1000, 2000, 3000 etc (Text ©PHP.net)

Delimited by - Set delimiter. Defaults to comma.

Enclosed by - Set enclosure character. Defaults to double quotes.

CSV Format

Each line must contain an answer and question. One per line. Example: "Question One","Answer One"
"Question Two","Answer Two"
"Question Three","Answer Three"
"Question Four","Answer Four"
"Question Five","Answer Five" Note that the default enclosure character and delimiter are used in this example. This may differ with your own data. Click the "Example CSV" link for a test CSV.

Categories

Use the checkboxes to specify which categories the imported questions apply to. If you click "Clear Existing" it will clear the current categories for the selected categories ONLY before the import is actioned.
