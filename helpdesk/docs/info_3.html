Below is an overview of the technology used to build Maian Support and other info.

Coding & Design

All Programming: Notepad++
Additional Graphics: Photo Filtre & AAA Logo
Icons: Mark James
Software Encryption: ionCube (Windows & Linux)
Searching: File Locator Pro
Screen Captures: Faststone Capture
MySQL Database Management: HeidiSQL
Notes, Ideas & Planning: ResophNotes syncing with Simplenote / Dropbox / Draft (Android)
Project Management: DevProject Manager
Versioning & Synchronisation: Syncovery & WinMerge
FTP: WinSCP

Script Version: 3.0
Released Date: Beta Version
Maian Support was conceived, designed & programmed by: David Ian Bennett
Maian Support website designed & created by: David Ian Bennett

Maian Support is a part of the Maian Script World software suite.

Operating Systems: Windows 7 & Windows 8.1 (Development), Linux Centos5 (Production)
Error Reporting Level for Development: E_ALL & ~E_DEPRECATED & ~E_STRICT
