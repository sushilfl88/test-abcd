Overview

Enables visitor to create ticket. This option can be disabled in the settings. It can also be set to only allow visitors to open tickets if they are logged in. If they are not logged in and the create tickets option is enabled, an account will be auto created. Please read carefully. Click example images to display full size version in new tab/window.

Screenshot

Basic Options

Your Name - Name or alias. This option is NOT show if a visitor is logged in.

E-Mail Address - Valid email address. This option is NOT show if a visitor is logged in.

Subject - Subject text for new ticket.

Department - Specify department. If applicable, may load custom fields (see below).

Priority - Specify priority.

Comments - Enter comments. BB code may be used to enhance formatting if enabled. More info on BB code here.

Custom Fields

If custom fields are set, they will load when a visitor selects a relevant department. See the following screenshot:

Attachments

Attachment boxes will display if attachments are enabled. If restriction are set the restrictions will be shown below the boxes. To add/remove boxes a visitor must click the +/- buttons. "Browse" is used to locate file on computer. See first screenshot.

reCaptcha

If enabled in settings, a reCaptcha box will appear to help prevent spam. This can be set to not appear if a visitor is logged in via the settings.

Preview Message

Once a visitor has typed some text into the textarea they can use this option to preview the post. This will format any BB code used if enabled for visitors. This screenshot shows the YouTube BB formatting when the preview option is used.



To clear the preview click on the text area text or use the close button.
