Overview

This is the public version of the F.A.Q system. Please read carefully. Click example images to display full size version in new tab/window if applicable.

Access F.A.Q

The F.A.Q system is accessed via the category links in the left menu.

Questions

Clicking a category link will show the questions for that category.

Search Questions

Click the icon to display the search box. You can skip common words from frontend searches via the following file:

control/skipwords.php

Viewing Question



Voting area will display if voting is enabled in settings. Nothing else displays on the frontend, this is for backend reference only.

If attachments are present, they will display beneath the question.

The printer friendly version utilises the "Print Friendly" browser plugin. More info here. Code can be updated or removed in the following file:

content/*theme*/faq-question.tpl.php
