Filters

The filters allow you to filter on screen data. Alternatively, click the icon to display the search box.

Ticket Info

The ticket view list is minimal for a clean interface. The list consists of "ID/Priority", "Subject/Category" and start and last reply information. A more detailed view of a ticket is seen when clicking the ticket number to view a ticket. More info on the ticket interface here.

Hover Icons

When hovering over a ticket row, icons will appear (seen in above screenshot). These are the edit and notepad icons. The notepad icon will ONLY display if a staff member has access to the notepad.
