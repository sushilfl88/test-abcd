Auto Close Tickets

  • Visitor Account /
  • Auto Close Tickets
If enabled in the admin settings, the system will auto close tickets that have been waiting a visitor reply, but have had no reply for x amount of days. This can help maintain a more efficient help desk system.

Optional email notification can also be sent to notify visitor of closed tickets. Visitor can always re-open tickets if required. For this to be automated, you need to set up a cron job (or cron tab) to execute the 'close-tickets.php' file on your server.

Manual Run

You can run the auto close routine via the admin settings at any time.
