Basic Search

Keyword (Name,E-Mail,Notes) - Enter keywords. Searches name, email and notes fields for match.

Date / Time Options

Date Added (From/To) - Looks for accounts opened between date range. Clicking into boxes will envoke the calendar for date selection.

Timezone - Looks for any accounts with the chosen timezone.

Filters

Account Status - If selected, looks for accounts for chosen status.

Amount of Tickets Opened is Between - Looks for any accounts that have x amount of tickets between count range.

Screenshot - Search Results

Filters

The filters allow you to filter on screen data.

Delete

To delete any options, click the relevant checkboxes and then click the "Delete Selected" button. A prompt will appear asking you to confirm your action. Note that deletions cannot be undone once actioned. If permissions aren`t sufficient, this option may be hidden.

Export to CSV

Enables you to export accounts to CSV. You do NOT need to select any accounts when using the export option. The export will always mirror the search results, so if you want specific accounts in the CSV, filter by search options first. Once you have the accounts you need use the "Export to CSV" button.

Tick / Disp

Displays the ticket and dispute counts. Dispute counts are only shown if the dispute system is enabled in your settings.

Update Search Parameters / Cancel Search

The update search parameters option lets you modify existing search params. The cancel option returns the search screen to its original status.
